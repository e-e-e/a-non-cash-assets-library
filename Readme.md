## running

```sh
# install gulp globally for dev
sudo npm i gulp -g

# install dependancies
npm install
gulp

# run server
node index.js

# or run server with gulp to automatically lint, 
# precompile and relaunch server on changes
gulp run

```

requires config.json:

```js
{
	"port": 8080
}
```
